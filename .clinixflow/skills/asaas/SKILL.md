# Instruções de Uso

Esta skill garante que nós, como assistentes, utilizemos o servidor MCP do Asaas para realizar integrações de cobrança, clientes e pagamentos.

- **Servidor MCP**: O servidor `asaas` foi configurado e deve estar disponível nas ferramentas (tools) quando interagimos com o projeto.
- Sempre que houver necessidade de listar clientes, enviar cobranças ou ler a documentação da API do Asaas, lembre-se de conferir se o MCP expõe uma ferramenta adequada e a utilize para poupar tempo.
